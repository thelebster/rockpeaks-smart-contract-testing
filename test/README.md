### Run tests

```
cd test

npm install

npm run test

# Run a single test by name
npm test -- -t "rp-clip-card"

# Run a single test file
npm test -- clip-card.test.js
```

* [JavaScript Testing Framework for Flow Network](https://github.com/onflow/flow-js-testing)
